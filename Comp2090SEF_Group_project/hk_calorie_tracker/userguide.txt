HK Calorie Tracker - Quick User Guide

What it does
Track calories, exercise, water, and health goals with an AI health coach.

Requirements
- Python 3.10+, 2GB RAM

Setup
1. cd hk_calorie_tracker
2. python -m venv .venv
3. Activate: .venv\Scripts\activate  (Windows) / source .venv/bin/activate  (macOS/Linux)
4. python -m pip install -r requirements.txt

*** Running the App ***
Just run the frontend - that's all you need:
   python run_frontend.py
Opens at http://localhost:8501  |  No separate backend needed.

Getting Started
- Create a profile: enter Name, Age, Weight, Height, Goal -> Click "Create Profile"
- Dashboard shows: Calorie Ring, Daily Stats, Streak, Tips

Key Features
- Food Diary : Search food -> Select -> Enter quantity -> Choose meal type -> "Log Food"
- Exercise   : Go to Exercise tab -> Choose activity -> "Log Exercise"
- Water      : Go to Water tab -> Enter glasses -> "Log Water"
- Goals/BMI  : Set calorie target -> "Save Calorie Goal" | Enter height/weight -> "Calculate BMI"
- Insights   : 7-day trend chart, consistency %, data export (text/CSV)
- AI Coach   : Ask nutrition or fitness questions; use quick buttons (Calories, Protein, Meal idea, Summary)
- Profiles   : Create, switch, edit, or delete profiles from the sidebar

Data
- Auto-saved to tracker_state.db (local, no internet required)
- Export: daily summary (text) or weekly history (CSV)
- Reset day: "Reset Day" button clears today's food logs

Troubleshooting
- App not opening?  -> Re-run: python run_frontend.py
- Port conflict?    -> set HK_TRACKER_API_PORT=5051  then run again
- Data not loading? -> Restart the app; check tracker_state.db exists
